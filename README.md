# Compilação

## Linux:

```sh
cmake ./
make
```
## Windows:
```sh
cmake -G "MinGW Makefiles" ./
make
```
# Uso:

```sh
Como usar:
    ./JVM <command> <package/class>
Comandos:
    leitor_exibidor
    jvm
Exemplos:
    ./JVM leitor_exibidor Main
    ./JVM jvm Main
```
